## Implementación de un login y registro con la utilización de los servicios de Firebase

## ⏬ Instalacion

Clona el proyecto

```bash
  git  clone https://github.com/martinizin/flutterChat.git
```

Posiciónate la carpeta del proyecto

```bash
  cd my-project
```

Instala las dependencias

```bash
  flutter pub get
```


Inicia el servidor

```bash
  flutter run
```
